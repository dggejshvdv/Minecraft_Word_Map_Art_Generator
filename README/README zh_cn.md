# 我的世界文字地图画生成器

一个用于生成 Minecraft 地图画投影文件的工具，支持将文字渲染为 128×128 的巨型平铺地图画，使用 Litematica 投影模组进行建造。

## 功能特点

- 🎨 **支持 16 种 Minecraft 混凝土颜色**：白色、橙色、品红色、淡蓝色、黄色、黄绿色、粉红色、灰色、淡灰色、青色、紫色、蓝色、棕色、绿色、红色、黑色
- ✍️ **自定义字体支持**：将字体文件放入 `ttf/` 目录即可使用
- 📝 **多字符批量生成**：一次性输入多个文字或符号，每个单独生成投影文件
- 🔤 **符号中文命名**：自动将符号转换为中文名称（如 `©` → `版权符号`）
- 📍 **角落定位符号**：特殊符号自动放置在地图角落（如 `©®™` 在右上角，`★☆` 在左上角）
- 📦 **自动创建目录**：运行时自动创建 `ttf/` 和 `litematic/` 目录
- 🎭 **图形界面**：使用 PyQt6 构建的现代化 GUI，操作更便捷
- 🌐 **多语言支持**：支持中文和英文，自动检测系统语言

## 安装

### 环境要求

- Python 3.8+
- 依赖库：`freetype-py`, `Pillow`, `litemapy`, `PyQt6`

### 安装依赖

```bash
pip install freetype-py Pillow litemapy PyQt6
```

### 字体准备

将字体文件（.ttf 格式）放入 `ttf/` 目录。工具会自动检测目录中的字体文件（最多 10 个）。

## 使用方法

### 启动程序

```bash
python ui.py
```

### 交互流程

1. **选择字体**：从下拉菜单中选择 `ttf/` 目录下的字体
2. **输入文字**：在文本框中输入要生成的文字或符号（支持多个字符）
3. **选择颜色**：点击颜色按钮选择文字颜色
4. **选择符号**：从符号面板中点击符号直接插入
5. **生成**：点击"生成投影文件"按钮

## 在游戏中使用

1. 将生成的 `.litematic` 文件放入 `.minecraft/schematics` 文件夹
2. 在游戏中安装 **Litematica** 投影模组（需前置 MaLiLib）
3. 使用木棍打开投影菜单，加载此文件
4. 将投影放置在地图区域内即可建造

## 文件说明

### ui.py

图形界面主程序，使用 PyQt6 构建，包含：
- 字体选择下拉菜单
- 文字输入框
- 颜色选择面板（16种混凝土颜色）
- 符号选择面板（分类展示常用符号）
- 输出日志显示区域
- 清除日志按钮
- 语言切换功能

### main.py

核心逻辑模块，负责：
- 字体支持检查
- 文字渲染为图像
- 图像转换为 Litematica 投影文件

**注意**：`main.py` 不再提供命令行模式，只能通过 `ui.py` 启动。

### i18n.py

国际化框架，支持：
- 自动检测系统语言
- 运行时切换语言
- JSON 格式翻译文件

### i18n/

翻译文件目录，包含：
- `zh_cn.json` - 中文翻译
- `en.json` - 英文翻译

## 技术说明

### 文件命名格式

`字体名称_字符名_颜色_方块数量.litematic`

例如：`AaZhuNiWoMingMeiXiangChunTian-2_你_红色_1234.litematic`

### 角落符号位置

| 符号    | 位置  |
| ----- | --- |
| `©®™` | 右上角 |
| `°★☆` | 左上角 |
| `§`   | 右下角 |
| `●○`  | 左下角 |

### 字体支持检查

工具使用 `freetype-py` 检查字体是否支持输入的字符。如果字体不支持某个字符，会提示用户重新输入。

### 语言切换

- 第一次启动时自动读取系统首选语言
- 在界面右上角可随时切换语言（中文/English）
- 切换语言后界面文字即时更新
