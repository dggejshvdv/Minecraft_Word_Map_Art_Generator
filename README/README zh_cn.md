# 我的世界文字地图画生成器

一个用于生成 Minecraft 地图画投影文件的工具，支持将文字渲染为 128×128 的巨型平铺地图画，使用 Litematica 投影模组进行建造。

## 功能特点

- 支持 16 种 Minecraft 颜色方块：混凝土、羊毛、羊毛地毯、陶瓦
- 系统字体支持：自动读取系统已安装的字体，无需手动放置字体文件
- 多字符批量生成：一次性输入多个文字或符号，每个单独生成投影文件
- 符号中文命名：自动将符号转换为中文名称（如 `©` → `版权符号`）
- 角落定位符号：特殊符号自动放置在地图角落（如 `©®™` 在右上角，`★☆` 在左上角）
- 自动创建目录：运行时自动创建 `ttf/` 和 `litematic/` 目录
- 图形界面：使用 PyQt6 构建的现代化 GUI，操作更便捷
- 多语言支持：支持中文和英文，自动检测系统语言
- 文件预览：双击生成的投影文件可打开预览窗口
- 两种预览模式：地图画预览、建造预览(2D)
- 材质渲染：支持使用 Minecraft 材质文件进行预览
- 设置功能：可配置默认方块类型、默认预览方式、默认字体

## 安装

### 环境要求

- Python 3.8+
- 依赖库：`freetype-py`, `Pillow`, `litemapy`, `PyQt6`

### 安装依赖

```bash
pip install freetype-py Pillow litemapy PyQt6
```

### 字体准备

无需手动放置字体文件！工具会自动读取系统已安装的字体。用户只需双击字体文件进行安装即可。

### 材质准备（可选）

如需使用材质渲染预览，需从 Minecraft 的 jar 文件中提取材质：

1. 找到 Minecraft 游戏目录下的版本 jar 文件（如 `1.21.jar`）
2. 使用解压工具打开 jar 文件
3. 将 `assets` 文件夹复制到项目根目录
4. 确保路径为：`assets/minecraft/textures/block/`

工具支持以下材质文件：
- `*_concrete.png` - 混凝土材质
- `*_wool.png` - 羊毛材质
- `*_carpet.png` - 羊毛地毯材质
- `*_terracotta.png` - 陶瓦材质

## 使用方法

### 启动程序

```bash
python ui.py
```

### 交互流程

1. 选择字体：从下拉菜单中选择系统已安装的字体
2. 输入文字：在文本框中输入要生成的文字或符号（支持多个字符）
3. 选择方块类型：从下拉菜单中选择方块类型（混凝土、羊毛、羊毛地毯、陶瓦）
4. 选择颜色：点击颜色按钮选择文字颜色
5. 选择符号：从符号面板中点击符号直接插入
6. 生成：点击"生成投影文件"按钮
7. 预览：双击右侧文件列表中的文件打开预览窗口

### 预览功能

生成投影文件后，双击文件列表中的文件可打开预览窗口：

- **地图画预览**：显示像素化的地图画样式（默认）
- **建造预览(2D)**：使用方块材质进行 2D 渲染，支持鼠标滚轮缩放和拖拽平移

**预览操作**：

- **鼠标左键拖拽**：平移视图
- **鼠标滚轮**：缩放视图（以鼠标位置为中心）
- **工具栏按钮**：放大、缩小、重置视图

### 设置

点击顶部菜单栏"设置 (Settings)"直接打开设置窗口：

- **默认方块类型**：设置启动时默认选中的方块类型（混凝土、羊毛、羊毛地毯、陶瓦）
- **默认预览方式**：设置预览窗口默认使用的预览模式（地图画、2D）
- **默认字体**：设置启动时默认选中的字体

设置保存在 `config.json` 文件中，包含设置和语言配置两部分。

## 在游戏中使用

1. 将生成的 `.litematic` 文件放入 `.minecraft/schematics` 文件夹
2. 在游戏中安装 **Litematica** 投影模组（需前置 MaLiLib）
3. 使用木棍打开投影菜单，加载此文件
4. 将投影放置在地图区域内即可建造

## 文件说明

### ui.py

图形界面主程序，使用 PyQt6 构建，包含：
- 顶部菜单栏（设置、语言）
- 字体选择下拉菜单
- 文字输入框
- 方块类型选择器
- 颜色选择面板（16种颜色）
- 符号选择面板（分类展示常用符号）
- 生成的文件列表（双击打开预览）
- 输出日志显示区域
- 清除日志按钮

### main.py

核心逻辑模块，负责：
- 字体支持检查
- 系统字体搜索
- 文字渲染为图像
- 图像转换为 Litematica 投影文件
- 支持多种方块类型（混凝土、羊毛、羊毛地毯、陶瓦）

注意：`main.py` 不再提供命令行模式，只能通过 `ui.py` 启动。

### i18n.py

国际化框架，支持：
- 自动检测系统语言
- 运行时切换语言
- JSON 格式翻译文件

### settings.json

设置文件，保存用户配置：
- 默认方块类型
- 默认预览方式
- 默认字体

## 技术说明

### 文件命名格式

`字体名称_字符名_方块类型_颜色_方块数量.litematic`

例如：`AaZhuNiWoMingMeiXiangChunTian-2_你_混凝土_红色_1234.litematic`

### 角落符号位置

| 符号    | 位置  |
| ----- | --- |
| `©®™` | 右上角 |
| `°★☆` | 左上角 |
| `§`   | 右下角 |
| `●○`  | 左下角 |

### 字体支持检查

工具使用 `freetype-py` 检查字体是否支持输入的字符。如果字体不支持某个字符，会提示用户重新输入。

### 语言切换

- 第一次启动时自动读取系统首选语言
- 在顶部菜单栏"语言 (Language)"中可切换语言（中文/English）
- 切换语言后界面文字即时更新
